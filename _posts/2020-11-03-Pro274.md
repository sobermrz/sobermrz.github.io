# 代码

```java
class Solution {
    public int hIndex(int[] citations) {
        //ans: Solution
        Arrays.sort(citations);

        int i = 0;
        int len = citations.length;
        while(i < len && citations[len - i - 1] > i){
            i++;
        }

        return i;//after loop, i comes to i+1
    }
}
```
