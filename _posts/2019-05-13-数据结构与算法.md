# 代码实现

## Node类
```java
package demo2;

public class Node {
	//节点内容
	int data;
	
	//下一个节点
	Node next;
	
	public Node(int data){
		this.data = data;
	}
	
	//为节点追加节点
	public Node append(Node node){
		//当前节点
		Node currentNode = this;
		//循环找
		while(true){
			//取出下一个节点
			Node nextNode = currentNode.next;
			//如果下一个节点为null,当前节点已经是最后一个节点
			if(nextNode == null){
				break;
			}
			
			//将下一个节点赋给当前节点
			currentNode = nextNode;
		}
		//把需要追加的节点追加为找到的当前节点的下一个节点
		currentNode.next = node;
		return this;
	}
	
	//获取下一个节点
	public Node next(){
		return this.next;
	}
	
	//获取内容
	public int getDate(){
		return this.data;
	}
	
	//当前节点是否为最后一个节点
	public boolean isLast(){
		return next == null;
	}
}


```

## Node测试类
```java
package demo2.test;

import demo2.Node;

public class TestNode {

	public static void main(String[] args) {
		//创建节点
		Node n1 = new Node(1);
		Node n2 = new Node(2);
		Node n3 = new Node(3);
		
		//追加节点
		n1.append(n2).append(n3);
		
		//取出下一个节点
		System.out.println(n1.next().next().getDate());
		
	}

}

```
